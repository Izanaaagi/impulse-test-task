# Impulse Test Task

## Installation

### Prerequisites
- Docker 19.03.0+

### Setup
1. Clone the repository
```shell
git clone https://github.com/Izanaaagi/impulse-test-task.git
```
2. Copy the environment configuration file
```shell
cp .env.example .env
```
3. Run the project
```shell
docker-compose up
```
